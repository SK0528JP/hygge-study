How to use Hygge Study

Hygge Studyは、GitHub Gistをデータベースとして利用する、サーバーレスな学習支援システムです。
心地よいデザインの中で、あなたの努力を記録・可視化しましょう。

1. Setup (初回接続)

データを保存するために、GitHubの「Personal Access Token」が必要です。

Step A: GitHub Token設定 へアクセス。

Step B: Generate new token (classic) を選択。

Step C: 権限の gist にチェックを入れて発行。

Step D: 発行されたトークンをコピーし、アプリの CONNECT GITHUB から入力。

⏱️ Timer

タイマーをクリックで開始/停止。停止時に学習データが自動的にクラウドへ保存されます。

📊 Charts

1日の目標時間（5時間）に対する進捗をリアルタイム表示。やる気を視覚化します。

📝 Task Plan

タスクの追加・削除・完了管理が可能。完了したタスクは下に移動します。

🎓 Goal Settings

Settingsから志望校と試験日を設定。試験までの残り日数を自動計算します。

Privacy & Security

トークンはあなたのブラウザのみに保存されます。
公共のPCを使用する場合は、終了時に必ずログアウトしてください。
