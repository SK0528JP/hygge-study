# 🌿 Hygge Study (ヒュッゲ・スタディ)

**Hygge Study** は、北欧デザインの「心地よさ」と GitHub の「機能性」を融合させた、サーバーレスな学習支援ウェブシステムです。
日々の学習時間の記録、スケジューリング、そして受験に向けたデータの可視化を、すべてあなたの GitHub アカウントのみで完結させます。

---

## ✨ 主な機能

- **北欧風モダン UI**: 集中力を高める、目に優しいミニマルなデザイン。
- **GitHub データベース**: 学習データはすべて自身の GitHub Gist にセキュアに保存されます。
- **学習タイマー**: ワンクリックで計測開始。停止時に自動でクラウドへ同期。
- **進捗の可視化**: 今日の目標達成度や試験までのカウントダウンを表示。

---

## 🚀 はじめに（セットアップ）

このシステムを動かすには、GitHub の **Personal Access Token (PAT)** が必要です。

### 1. トークンの発行
GitHub でデータを保存するための「鍵」を作成します。
1. GitHub の [Settings > Developer settings > Personal access tokens (classic)](https://github.com/settings/tokens) へアクセス。
2. **"Generate new token (classic)"** を選択。
3. Note に「Hygge Study Data」などと入力。
4. **スコープ（権限）の `gist` にチェックを入れる。**
5. 下部の "Generate token" を押し、表示されたトークンをコピーします（※一度しか表示されないので注意）。

### 2. システムへの接続
1. 公開したウェブサイトを開きます。
2. 右上の **"LOGIN WITH GITHUB"** ボタンをクリック。
3. 先ほどコピーしたトークンを貼り付けます。
4. 「学習用データベースを作成しました」と表示されれば準備完了です！

---

## 📖 使い方

- **学習を始める**: 画面左側の「学習時間（00:00:00）」部分をクリックするとタイマーが始動します。
- **記録を保存する**: 学習が終わったら、もう一度時間をクリックしてタイマーを止めます。データは自動的に GitHub へ保存されます。
- **計画を立てる**: 右側の Study Plan セクションで、やるべきタスクを確認しましょう。

## 🎨 デザインコンセプト: Hygge (ヒュッゲ)
デンマーク語で「心地よい時間や空間」を意味する *Hygge*。
受験勉強や日常の学習という、時には過酷な時間を、少しでも穏やかで前向きなものに変えるために、余白と色使いにこだわっています。
