# 🌿 Hygge Study (ヒュッゲ・スタディ)
> **GitHub Gist をデータベースにした、北欧風デザインの受験戦略・学習支援システム**

Hygge Study は、デンマーク語で「心地よい時間」を意味する **"Hygge (ヒュッゲ)"** をコンセプトにした学習管理ツールです。
サーバーを一切介さず、あなたの GitHub アカウント（Gist API）をプライベートなデータベースとして利用することで、セキュアかつ永続的に学習記録を管理します。

---

## ✨ 主な機能

### 1. ⏱️ フォーカス・タイマー
- 画面中央のタイマーをクリックするだけで、学習セッションを開始。
- 稼働中は北欧ブルーの光が優しく点滅（パルスアニメーション）。
- 停止すると、そのセッションの学習時間を自動的に GitHub へ保存。

### 2. 📊 学習データの可視化 (Chart.js)
- 1日の目標時間（デフォルト5時間）に対する進捗をドーナツグラフで表示。
- 勉強した分だけグラフがセージグリーンに染まり、達成感を視覚化。

### 3. 📝 受験戦略タスク管理
- 学習タスクを自由に追加・削除。
- 完了したタスクはチェックを入れると斜線が引かれ、リストの下部へ移動。
- すべての変更は GitHub 上の JSON ファイルに即座に同期。

### 4. 🎓 志望校カウントダウン
- 設定画面から「志望校（目標）」と「試験日」を入力可能。
- 試験までの残り日数を自動計算し、メイン画面に表示。

---

## 🚀 クイックスタート (セットアップ)

このシステムはサーバーレスのため、以下の手順で「自分専用のデータベース」を接続する必要があります。

### STEP 1: GitHub トークンの発行
1. [GitHub Personal Access Tokens (classic)](https://github.com/settings/tokens) にアクセス。
2. **"Generate new token (classic)"** をクリック。
3. **Note**: `Hygge Study Token` など。
4. **Select scopes**: **`gist`** にチェックを入れる（これだけでOK）。
5. 下部の "Generate token" を押し、表示されたトークンをコピー。
   - ※一度しか表示されないので大切に保管してください。

### STEP 2: システムへの接続
1. デプロイした `index.html`（またはローカル環境）を開きます。
2. 右上の **"CONNECT GITHUB"** ボタンをクリック。
3. コピーしたトークンを貼り付けて実行。
4. 成功すると、あなたの GitHub Gist 内に `hygge_data.json` というファイルが自動生成されます。

## 📖 使い方

- **学習を記録する**: タイマーカードをクリックして開始。終わったら再度クリックして停止。
- **目標を立てる**: 右上の `Settings` をクリックし、志望校と試験日を入力して保存。
- **タスクをこなす**: `+ ADD TASK` でやるべきことを追加。終わったらチェック。
- **ログアウト**: 接続済みのボタンをクリックすると、ローカルの認証情報を削除できます。

## 🎨 デザインコンセプト

- **Color Palette**: Nord Theme (Snow Storm, Frost, Aurora)
- **Typography**: Montserrat & Noto Sans JP
- **Glassmorphism**: 16pxの背景ぼかしと透過による、目に優しく洗練された質感。

集中力を削ぐ過度な装飾を排除し、余白と色使いによって「勉強したくなる空間」を目指しています。

---

## 🛠 技術スタック

- **Frontend**: HTML5 / CSS3 (Tailwind CSS)
- **Library**: [Chart.js](https://www.chartjs.org/) (Data Visualization)
- **Backend/DB**: [GitHub Gist API](https://docs.github.com/en/rest/gists)
- **Hosting**: GitHub Pages 推奨

---

## ⚠️ 注意事項

- 本システムはフロントエンドから GitHub API を直接叩きます。トークンはブラウザの `localStorage` にのみ保存されますが、公共のPCで使用する場合は、必ず使用後にログアウト（Clear Storage）を行ってください。
- データの保存先である Gist を手動で削除すると、システムの同期ができなくなります。その場合はトークンを再入力して再接続してください。
